# Build Your First React.js App

Asciicasts for [Dan Abramov](https://github.com/gaearon) course on [egghead.io](https://egghead.io/courses/building-react-applications-with-idiomatic-redux).

## Gitbook
```
npm i -g gitbook-cli
npm install
gitbook install
gitbook serve
```
